Introducation to CSS Text Properties

CSS provides several properties for styaling and formatting text. This tutorial covers the most commonly used text properties including color, alignment,decoration, transformation, spacing, and shadow effects.

Text Color

The color property is used to set the color of text

This text is colored using the hex code #ff6600

CSS code: color: #ff6600

Text Alignment

The text-align property is used to set the horizontal alignment of text

This text is aligned to the left (default)

CSS code: text-align: left;

Text Decoration

The text-decoration property is used to add or remove decorations from text

This text has an underline.

CSS code: text-decoration: underline;

This text has an overline

CSS code: text-decoration: overline;

This text has a line through it.

CSS code: text-decoration: line-through;

Links typically have an underline by default, but it can be removed:

This link has no underline

CSS code: text-decoration: none;

Text Transformation

The text-transform property is used to change the capitalization of text.

This text is transformed to uppercase

CSS code: text-transform: uppercase;

This text is transformed to lowercase

CSS code: text:transform: lowercase;

This text is transformed to capitalize each word

CSS code: text-transform: capitalize;

Text Spacing

CSS provides properties to control the spacing between letters, words, and lines.

This paragraph demonstrates letter spacing, word spacing, and line height. Notice how the letters have more space between them, the words are further apart, and there's more vertical space between lines.

CSS code

Letter-spacing: 2px

Word-spacing: 5px

Line-height: 2;

Text Shadow

The text-shadow property adds shadow effects to text.

This text has a basic shadow effect.

CSS code: text-shadow:2px 2px 4px #888;

This text has a glow effect

CSS code: text-shadow: 0 0 5px #ff6600, 0 0 10px #ff6600
